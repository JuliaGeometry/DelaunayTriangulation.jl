Point-in-Polygon Testing

We have a function available for computing the distance from a point to a polygon. Using this, we are able to detect if a point is inside a polygon. To now determine which of these query points are inside any of the magenta regions, we check if the distance to the polygon is positive (inside) or negative (outside).

is_inside = [DelaunayTriangulation.distance_to_polygon(q, points, nodes) > 0 for q in query_points]
scatter!(ax, query_points[is_inside], color=:blue)
scatter!(ax, query_points[.!is_inside], color=:red)
fig

We see that we can correctly perform point-in-polygon tests this way. We also emphasise that this check is not using exact arithmetic, unlike other predicates in this package, so there may be some robustness issues here for points very close to the boundary.
